## ✨ 特性
- 所见即所得的开发体验
- 通用无代码，通过图形化编排的方式可实现复杂的功能
- 方便且强大的即时调试能力

##  ⌨️ 本地开发
- 可以下载PC端应用-https://mybricks.world/
- 或者clone到本地开发:
```
$ git clone git@github.com:mybricks/comlib-pc-normal.git
$ npm install
$ npm run start
```
